# AgendaOrg Backend

API Node.js + Express + TypeScript do AgendaOrg.

## Requisitos

- Node.js 22+
- npm
- MongoDB em `mongodb://127.0.0.1:27017/agendaorg`

## Comandos

```bash
npm install --cache .npm-cache
npm run seed
npm run dev
```

Comandos adicionais:

```bash
npm run build
npm run test
```

## URL

- API: http://localhost:3333/api
